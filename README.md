# 説明
ブックのプリンターデバイスを設定できる。

## 活用例
毎回ブック起動時にデフォルトのプリンターで印刷されてしまう手間を解消できる。

同じブックでも使用するPCによってプリンターが参照できない問題を解消できる。


# 使い方

## 設定

-  ModPrinter.bas
